Command leo

This is a starter project for a proxy to https://dialogflow.com/. Contains a main method that starts web service and assigns handlers to different endpoints.

Subdirectories

Name Synopsis
..
callers Contains structs and functionality to perform DialogFlow.com and Engineering.com api calls
chat Very basic chat.
listener Contains an api listener, its job is to process http requests.
models
caller_options Holds default values for DialogFlow API caller options
dialog_flow_objects Contains structs to model particular type of dialog flow object
queries Defines few structs to model corresponding dialogflow objects.
responses Contains the following structs: AgentQueryResponse, Result, Fullfilment that maps to dialogflow objects.
util Contains a collection of utility functions
Build version go1.10.
